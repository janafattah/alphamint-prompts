# Lead Magnet: Notion Prompt Library (Setup)

## Wofür ist das?
Erstellt Struktur, DB-Felder, Tags und Onboarding Copy für einen Notion Lead Magnet.

## Inputs (bitte ausfüllen)
- Brand/Projektname
- Zielgruppe
- Kategorien (AI 4 ...)
- Wie bekommen Nutzer Zugriff?
- CTA Ziel (Newsletter/Call)

## Prompt (kopieren)

Erstelle ein Notion Lead-Magnet Setup.

Lieferung:
- Datenbank-Schema (Felder + Typen)
- Tag-System
- Navigationsstruktur (Sidebar)
- Onboarding-Seite (Text)
- FAQ + Safety Hinweis
- 3 CTAs (ultra soft)

## Output-Format
- Schema + Seitenstruktur
- Onboarding Copy + FAQs + CTAs
