# Free Resource Hub

## Wofür ist das?
Plant eine kostenlose Resource-Seite (Notion) mit Navigation und Content-Plan.

## Inputs (bitte ausfüllen)
- Brand
- Themen-Säulen
- Formate (PDF, Notion, Video)
- Update-Frequenz
- CTA

## Prompt (kopieren)

Plane einen Resource Hub.

Lieferung:
- Seitenstruktur
- Kategorien
- Start-Content (10 Ideen)
- Update-Prozess (SOP)
- CTA Placement

## Output-Format
- Struktur + Content-Ideen + SOP
