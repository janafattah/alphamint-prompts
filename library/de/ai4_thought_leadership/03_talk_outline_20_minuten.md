# Talk Outline 20 Minuten

## Wofür ist das?
Baut eine 20-Min Talk-Struktur mit Story + Takeaways.

## Inputs (bitte ausfüllen)
- Thema
- Zielpublikum
- Dein Aha-Moment
- 3 Kernbotschaften
- Was sollen sie danach tun?

## Prompt (kopieren)

**Rolle:** Speaker Coach.

**Aufgabe:** Erstelle ein Talk-Outline.

**Struktur:**
- Opening (Hook)
- Story
- 3 Segmente
- Interaktion (1 Stelle)
- Closing + CTA

**Regeln:** Zeitboxing, keine Überfrachtung.

## Output-Format
- Talk Outline + Timing
