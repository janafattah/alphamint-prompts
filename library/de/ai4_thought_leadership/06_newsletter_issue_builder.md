# Newsletter Issue Builder

## Wofür ist das?
Schreibt eine Newsletter-Ausgabe im Jana-Ton.

## Inputs (bitte ausfüllen)
- Thema
- Kern-Insight
- 1 Story
- 1 Tool/Template
- CTA (soft)

## Prompt (kopieren)

**Rolle:** Newsletter Editor.

**Aufgabe:** Schreibe eine Ausgabe.

**Struktur:**
- Einstieg (Story)
- Insight
- Praktischer Teil (Tool/Template)
- Abschluss + CTA

**Regeln:** Kein Marketing-Overload.

## Output-Format
- Newsletter Text
