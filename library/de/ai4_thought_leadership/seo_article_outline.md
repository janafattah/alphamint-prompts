Erstellen Sie einen umfassenden SEO-optimierten Artikelentwurf und Content-Plan:

1. Generieren Sie eine umfangreiche Liste von LSI- und NLP-Keywords im Zusammenhang mit [THEMA]. Fügen Sie Synonyme, verwandte Begriffe und branchenspezifisches Fachvokabular hinzu.

2. Entwickeln Sie einen detaillierten Entwurf für einen 2.000-Wort-Artikel:
   • Erstellen Sie Hauptüberschriften und Unterüberschriften und stellen Sie eine hohe Keyword-Dichte sicher
   • Weisen Sie jedem Abschnitt Wortzahlen zu, insgesamt 2.000 Wörter
   • Fügen Sie eine Einleitung, einen Schluss und einen FAQ-Abschnitt hinzu
   • Stellen Sie sicher, dass der Entwurf umfassend genug ist, um einen vollständigen Artikel anzuleiten

3. Geben Sie für jeden Hauptabschnitt Folgendes an:
   • Wichtige zu behandelnde Punkte
   • Relevante Statistiken oder Daten, die einbezogen werden sollen
   • Potenzielle Beispiele oder Fallstudien

4. Erstellen Sie einen FAQ-Abschnitt mit 5-7 häufig gestellten Fragen zu [THEMA], basierend auf Suchtrends und Nutzerabsicht.

5. Schlagen Sie 3 relevante externe Links vor, die im Artikel enthalten sein sollen:
   • Geben Sie URLs und empfohlenen Ankertext an
   • Stellen Sie sicher, dass die Links von autoritativen Quellen stammen und keine direkten Konkurrenten sind

6. Empfehlen Sie interne Verlinkungsmöglichkeiten innerhalb des Artikels.

7. Geben Sie SEO-Optimierungstipps:
   • Vorgeschlagener Meta-Titel und Beschreibung
   • Ideen für Bild-Alt-Text
   • Empfehlungen zur Keyword-Platzierung

8. Skizzieren Sie eine Content-Distribution-Strategie:
   • Social-Media-Plattformen zum Teilen
   • Potenzielle Outreach-Ziele für Backlinks

9. Schreiben Sie den vollständigen 2.000-Wort-Artikel basierend auf dem Entwurf und den Empfehlungen.

10. Schlagen Sie 3-5 potenzielle Titel für den Artikel vor, optimiert sowohl für SEO als auch für Leserengagement.

Stellen Sie mir klärende Fragen, bis Sie zu 95% sicher sind, dass Sie die Aufgabe erfolgreich abschließen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, das Internet zu durchsuchen, um aktuelle Informationen abzurufen.
