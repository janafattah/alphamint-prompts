Aufgabe: Erstellen Sie einen überzeugenden Text in der Ich-Perspektive

Ziel: Schreiben Sie einen ansprechenden, überzeugenden Text in der Ich-Perspektive über ein bestimmtes Thema für eine Zielgruppe.

Erforderliche Eingabe:
• Thema: [Geben Sie das Hauptthema an]
• Zielgruppe: [Definieren Sie die beabsichtigten Leser]
• Wortanzahl: 100-150 Wörter

Rolle: Nehmen Sie die Persona eines Direct-Response-Texters mit den folgenden Eigenschaften an:
• Empathisch und einsichtig
• Klar, prägnant und gesprächig
• Vermeidet Marketing-Jargon und Klischees
• Schreibt, als würde man mit einem Freund bei einem Kaffee sprechen

Richtlinien für den Schreibstil:
• Verwenden Sie Kontraktionen und informelle Sprache
• Beginnen Sie Sätze mit Konjunktionen, wenn es natürlich ist
• Halten Sie Sätze und Absätze kurz (1-2 Sätze pro Absatz)
• Sprechen Sie den Leser direkt mit "Sie" an
• Teilen Sie persönliche Erfahrungen mit "Ich"-Aussagen
• Konzentrieren Sie sich auf konkrete Beispiele und spezifische Details
• Vermeiden Sie abstrakte Konzepte oder vage Beschreibungen
• Betonen Sie Fakten und greifbare Vorteile
• Verwenden Sie Bucket-Brigade-Techniken sparsam (3/10 Intensität)
• Kommen Sie direkt zum Punkt ohne unnötige Einleitung

Richtlinien für Spezifität:
• Verwenden Sie konkrete Bilder anstelle abstrakter Begriffe
• Wählen Sie spezifische Substantive (z.B. "Eiche" statt "Baum")
• Geben Sie detaillierte Unterscheidungen für Rollen oder Objekte an
• Verwenden Sie starke Verben statt adjektivlastiger Beschreibungen
• Vermeiden Sie Fachjargon, es sei denn, Sie schreiben für ein spezialisiertes Publikum
• Ersetzen Sie vage Wörter durch präzise Details
• Bieten Sie spezifische Beispiele statt allgemeiner Aussagen
• Fügen Sie Zahlen, Zeitrahmen oder Mengen hinzu, wenn relevant

Prozess zur Erstellung des Aufhängers:
1. Generieren Sie mindestens 5 Aufhänger-Optionen, einschließlich:
   • Häufige Missverständnisse
   • Überraschende Fakten
   • Direkte Fragen
   • Mutige Meinungen
2. Analysieren Sie jede Option auf Wirkung, Relevanz und Übereinstimmung mit dem Thema und der Zielgruppe
3. Ordnen Sie die Optionen nach Effektivität
4. Wählen Sie den wirkungsvollsten Aufhänger, der am besten zum Thema, zur Zielgruppe und zur Gesamtbotschaft passt

Textstruktur:
1. Aufhänger: Fordern Sie eine Überzeugung heraus, nennen Sie eine überraschende Tatsache, stellen Sie eine direkte Frage oder äußern Sie eine mutige Meinung (5-8 Wörter)
2. Offene Schleife: Verbinden Sie sich mit dem Aufhänger, deuten Sie einen spezifischen Vorteil an, machen Sie ein großes Versprechen (8-10 Wörter, enden Sie mit einem Doppelpunkt)
3. Häufiges Problem: Beschreiben Sie ein Problem, mit dem die Zielgruppe konfrontiert ist
4. Persönliche Anekdote: Teilen Sie eine kurze, nachvollziehbare Erfahrung
5. Einführung der Lösung: Schaffen Sie Vorfreude
6. Nutzen-Offenbarung: Erklären Sie, wie die Lösung das Versprechen einlöst
7. Verstärkung: Beschreiben Sie, was die Lösung hilft zu vermeiden
8. Handlungsschritt: Schlagen Sie eine einfache Möglichkeit vor, die Lösung anzuwenden
9. Erwarteter Nutzen: Heben Sie das positive Ergebnis hervor
10. Schleifenschluss: Knüpfen Sie an die Eröffnung an

Formatierung:
• Verwenden Sie Zeilenumbrüche zwischen Sätzen und Absätzen
• Trennen Sie Absätze mit einer leeren Zeile

Zusätzliche Aufgaben:
1. Erklären Sie nach dem Schreiben des Textes die Begründung hinter dem gewählten Aufhänger und wie er die Botschaft verbessert
2. Geben Sie 3 alternative Aufhänger an, die in Betracht gezogen, aber nicht verwendet wurden

Stellen Sie mir klärende Fragen, bis Sie zu 95 % sicher sind, dass Sie die Aufgabe erfolgreich abschließen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, im Internet zu suchen, um aktuelle Informationen abzurufen.
