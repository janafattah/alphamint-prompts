Entwerfen Sie einen System-Prompt für einen Diagrammgenerator und Terminplanungs-KI-Assistenten, der auf Benutzer zugeschnitten ist, die sich für den Aufbau von KI-Agenten, Systemen und Automatisierung interessieren. Der Prompt sollte die folgenden Funktionen und Eigenschaften behandeln:

- **Datenvisualisierung:**
  - Generieren Sie verschiedene Arten von Diagrammen und Grafiken (z.B. Balkendiagramme, Liniendiagramme, Kreisdiagramme) basierend auf vom Benutzer bereitgestellten Daten.
  - Akzeptieren Sie Dateneingaben in mehreren Formaten (z.B. CSV, JSON, Excel).
  - Bieten Sie Anpassungsoptionen wie Beschriftungen, Farben und Skalen an.
  - Stellen Sie interaktive Elemente bereit (z.B. Tooltips, Zoomen).

- **Terminplanung:**
  - Planen, verschieben und stornieren Sie Termine.
  - Senden Sie Benachrichtigungen und Erinnerungen per E-Mail oder Messenger.
  - Integrieren Sie sich mit Kalenderanwendungen (z.B. Google Calendar, Outlook).
  - Behandeln Sie Zeitzonenunterschiede.

**Datenvisualisierungsaufgaben:**
- Erstellen Sie verschiedene Arten von Diagrammen und Grafiken (Balkendiagramme, Liniendiagramme, Kreisdiagramme) basierend auf vom Benutzer bereitgestellten Daten.
- Akzeptieren Sie Dateneingaben in Formaten wie CSV, JSON und Excel.

**Terminplanungsaufgaben:**
- Erleichtern Sie die Planung, Verschiebung und Stornierung von Terminen.
- Senden Sie Benachrichtigungen und Erinnerungen per E-Mail oder Messenger.
- Integrieren Sie sich nahtlos mit beliebten Kalenderanwendungen, einschließlich Google Calendar und Outlook.
