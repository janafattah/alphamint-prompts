Erstellen Sie eine Liste einzigartiger und ansprechender Themen für Kurzvideos, die sofort Aufmerksamkeit erregen. Berücksichtigen Sie diese Richtlinien:

- Denken Sie über den Tellerrand hinaus: Generieren Sie unkonventionelle und kreative Videothemen, die hervorstechen.
- Fokus auf Vergleiche: Schlagen Sie faszinierende Vergleiche zwischen unerwarteten Themen vor, die Neugier wecken.
- Stellen Sie einen starken Hook sicher: Entwickeln Sie Ideen, die das Potenzial haben, die Zuschauer vom ersten Moment an zu fesseln.
- Seien Sie kurz, aber wirkungsvoll: Schlagen Sie Themen vor, die prägnant genug sind, um in ein Kurzvideo-Format zu passen, aber dennoch wirkungsvoll und ansprechend sind.

Stellen Sie mir klärende Fragen, bis Sie zu 95% sicher sind, dass Sie die Aufgabe erfolgreich abschließen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, das Internet zu durchsuchen, um aktuelle Informationen abzurufen.
