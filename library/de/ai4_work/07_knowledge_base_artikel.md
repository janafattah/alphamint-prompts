# Knowledge Base Artikel

## Wofür ist das?
Erstellt KB-Artikel, die wirklich helfen (nicht nur Doku).

## Inputs (bitte ausfüllen)
- Thema
- Zielgruppe
- Häufige Fehler
- Screenshots/Infos (optional)
- Do/Don't

## Prompt (kopieren)

**Rolle:** Technical Writer.

**Aufgabe:** Schreibe einen KB-Artikel.

**Struktur:**
- Problem
- Lösung in 5 Schritten
- Beispiele
- Troubleshooting
- FAQ (3)

**Regeln:** Einfaches Deutsch, keine Tool-Werbung.

## Output-Format
- KB Artikel
