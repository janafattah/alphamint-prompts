Erstellen Sie ein umfassendes Budget für ein Kinderprogramm basierend auf den folgenden Details:

- **Projektdauer**: Definieren Sie, wie viele Tage das Programm laufen wird.
- **Arbeitsstunden der Lehrkräfte**: Stellen Sie sicher, dass jede Lehrkraft täglich mindestens 4-5 Stunden arbeitet.
- **Gehalt der Lehrkräfte**: Berechnen Sie das Gesamtgehalt für Lehrkräfte basierend auf 22 $ pro Stunde.
- **Programmkosten**: Berücksichtigen Sie zusätzliche Kosten wie Materialien, Raummiete, Werbung, Versicherung und alle anderen potenziellen Ausgaben. Geben Sie ein geschätztes Budget für jede Kategorie an.
- **Budget-Format**: Organisieren Sie das Budget in einer klaren und strukturierten Weise für einfaches Verständnis und Verarbeitung.

Bitte geben Sie eine detaillierte Antwort, die jede Komponente des Budgets aufschlüsselt, und treffen Sie gegebenenfalls Annahmen.

Stellen Sie mir klärende Fragen, bis Sie zu 95 % sicher sind, dass Sie die Aufgabe erfolgreich erledigen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, im Internet zu suchen, um aktuelle Informationen abzurufen.
