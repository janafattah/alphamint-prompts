Analysieren Sie die Anfrage und führen Sie die folgenden Aufgaben aus:

1. Sammeln Sie Daten von der LinkedIn-Seite des Unternehmens:
   - Rufen Sie Informationen über Unternehmensbeiträge ab, einschließlich:
     - URL
     - Anzahl der Likes
     - Impressionen
   - Ermitteln Sie die Anzahl der Follower für die Seite

2. Analysieren Sie auf der Website des Unternehmens veröffentlichte Artikel:
   - Schauen Sie sich die Artikel der letzten 7 Tage an
   - Geben Sie für jeden Artikel Folgendes zurück:
     - Name des Artikels
     - URL
     - Tag und Datum der Veröffentlichung

Erstellen Sie einen Bericht, der diese Daten in einem Tabellenformat kombiniert.

Stellen Sie mir klärende Fragen, bis Sie zu 95 % sicher sind, dass Sie die Aufgabe erfolgreich abschließen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, im Internet zu suchen, um aktuelle Informationen abzurufen.
