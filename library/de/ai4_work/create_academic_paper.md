---

Sie haben die Aufgabe, eine Seminararbeit von 2000-2500 Wörtern zu schreiben, die sich auf ein Unternehmen konzentriert, das Sie auswählen werden. Die Arbeit sollte eine spezifische Rolle innerhalb des Unternehmens einnehmen, wie z. B. Manager, Digital Marketer, Organisator usw., abhängig von der Themenvorgabe. Stellen Sie sicher, dass Sie:

- **Ein Unternehmen auswählen**: Wählen Sie ein bekanntes Unternehmen, das ausreichend Daten und Einblicke für Ihre Arbeit liefern kann.
- **Rollenspezifikation**: Identifizieren Sie klar die Rolle, die Sie innerhalb des Unternehmens übernehmen, und erklären Sie, wie diese Perspektive Ihre Analyse beeinflusst.
- **Strukturierung der Arbeit**: Organisieren Sie die Arbeit mit einer klaren Einleitung, These, Hauptteil und Fazit. Fügen Sie Abschnittsüberschriften für Klarheit hinzu.
- **Alle Quellen zitieren**: Nutzen Sie wissenschaftliche Artikel, Bücher und seriöse Online-Quellen zur Unterstützung Ihrer Argumente.
- **APA-Formatierung**: Stellen Sie sicher, dass alle Zitate und die Bibliographie gemäß den neuesten APA-Standards formatiert sind. Fügen Sie Textverweise hinzu und geben Sie eine umfassende Referenzliste an.
- **Bewertung und Analyse**: Bieten Sie eine kritische Bewertung und Analyse der Unternehmensstrategie oder -operationen aus der Perspektive Ihrer gewählten Rolle.
- **Einhalten der Wortzahl**: Stellen Sie sicher, dass die Arbeit zwischen 2000-2500 Wörtern liegt und durchgehend Kohärenz und Relevanz beibehält.

Stellen Sie mir klärende Fragen, bis Sie zu 95 % sicher sind, dass Sie die Aufgabe erfolgreich erledigen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, im Internet zu suchen, um aktuelle Informationen abzurufen.
