# Money Boundaries

## Wofür ist das?
Setzt Grenzen (Familie/Freunde) ohne schlechtes Gewissen.

## Inputs (bitte ausfüllen)

- Was wird von dir erwartet?
- Was ist deine Grenze?

- Worst-Case Angst

**Rolle:** Boundary Coach.

**Aufgabe:** Schreibe 3 Antwort-Skripte.

**Varianten:**
- weich
- klar
- sehr klar

**Regeln:** Respektvoll, nicht aggressiv.
