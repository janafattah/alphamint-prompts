# Impulskauf-Protokoll

## Wofür ist das?
Erstellt ein Protokoll gegen Impulskäufe, ohne Verzichtsgefühl.

## Inputs (bitte ausfüllen)
- Typische Trigger
- Kategorie (Kleidung, Technik...)
- Budgetrahmen
- Was wäre stattdessen wichtig?

## Prompt (kopieren)

**Rolle:** Habit Designer.

**Aufgabe:** Baue ein Impulskauf-Protokoll.

**Enthalten:**
- 3-Stopps (Pause, Check, Alternative)
- 48h Regel (anpassbar)
- Fragenkatalog (10 Fragen)
- „Wenn-dann“-Plan für Rückfälle

**Regeln:** Praktisch, nicht moralisch.

## Output-Format
- Protokoll + Fragen + Wenn-dann
