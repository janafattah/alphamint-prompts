# Zielbild: finanziell frei, aber real

## Wofür ist das?
Formuliert ein realistisches Zielbild + Zwischenstufen.

## Inputs (bitte ausfüllen)
- Was bedeutet Freiheit für dich?
- Zeithorizont
- Aktueller Zustand (kurz)
- Risiko-Toleranz
- Familie/Verpflichtungen

## Prompt (kopieren)

**Rolle:** Strateg:in.

**Aufgabe:** Formuliere ein Zielbild + Meilensteine.

**Lieferung:**
- Zielbild (Absatz)
- 5 Meilensteine
- Für jeden: 1 Kennzahl + 1 Verhalten

**Regeln:** Kein „Hustle“. Klar, menschlich.

## Output-Format
- Zielbild + Meilensteine
