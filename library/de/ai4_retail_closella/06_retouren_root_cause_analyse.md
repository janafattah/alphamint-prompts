# Retouren-Root-Cause Analyse

## Wofür ist das?
Analysiert Retourengründe und leitet Fixes ab (für Shops/Brands).

## Inputs (bitte ausfüllen)
- Retouren-Daten (Gründe, Kategorien)
- Top Produkte
- Kundensegmente
- Aktuelle Produktseiten
- Ziel (Retouren -X%)

## Prompt (kopieren)

**Rolle:** Retail Analyst.

**Aufgabe:** Erstelle eine Root-Cause Analyse.

**Lieferung:**
- Pareto: Top 3 Ursachen
- Hypothesen + Tests
- Fixes: Produktseite, Größeninfo, Fotos, Support
- KPI Set

**Regeln:** Keine Fantasie-Zahlen.

## Output-Format
- Analyse + Maßnahmenplan + KPIs
