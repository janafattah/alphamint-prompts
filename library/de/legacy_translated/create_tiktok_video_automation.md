- Erstellen Sie eine Liste von 100 Self-Help-Phrasen, die auf TikTok oder Instagram die meisten Kommentare und Weiterleitungen haben.
- Entwickeln Sie eine Methode, um jede Phrase automatisch mithilfe von Künstlicher Intelligenz in ein Video umzuwandeln.
- Speichern Sie die erstellten Videos in einem dafür vorgesehenen Ordner auf der lokalen Festplatte.
- Implementieren Sie ein Planungssystem, das täglich eines der gespeicherten Videos auf einem TikTok-Konto veröffentlicht.
- Stellen Sie sicher, dass der gesamte Prozess von Anfang bis Ende automatisiert ist.
- Identifizieren Sie alle Schritte, die nicht automatisiert werden können, informieren Sie den Benutzer und geben Sie detaillierte Anweisungen für die manuelle Durchführung dieser Schritte.
- Geben Sie Schritte zur Verbesserung der Gesamteffizienz des Prozesses an.

Stellen Sie mir klärende Fragen, bis Sie zu 95 % sicher sind, dass Sie die Aufgabe erfolgreich erledigen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, im Internet zu suchen, um aktuelle Informationen abzurufen.
