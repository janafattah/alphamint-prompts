# AI-4 Prompt-System (DE)

Das AI-4 System strukturiert Prompts nach vier Lebens- und Arbeitsbereichen, damit Inhalte klar auffindbar und konsistent bleiben:
- Enablement: Befähigung, Skills, Teams
- Work: Arbeitsprozesse, Produktivität, Qualität
- Wealth: Finanzen, Vermögen, langfristige Ziele
- Financial Wellbeing: Alltag, Budget, Entscheidungen im Kleinen

## So nutzt du die Prompts
1. Öffne den passenden Prompt.
2. Fülle die Inputs oben in der Datei aus (so konkret wie möglich).
3. Starte die Ausführung und erwarte den Output im angegebenen Format.
4. Prüfe das Ergebnis und ergänze fehlenden Kontext.
