### ChatGPT Prompt

Ich habe eine Anfrage zur Verbesserung einer Website. Könnten Sie bitte:

1. Eine Suchfunktion für die Website entwerfen, die es Benutzern speziell ermöglicht, nach vorhandenen Prompts zu suchen?
2. Die technischen Schritte oder Technologien beschreiben, die zur Implementierung dieser Suchfunktion erforderlich sind.
3. Best Practices zur Optimierung der Suchfunktion für Leistung und Benutzererfahrung vorschlagen.
4. Erklären, wie sichergestellt werden kann, dass die Suchfunktion sicher ist und Benutzerdaten schützt.

Stellen Sie mir klärende Fragen, bis Sie zu 95% sicher sind, dass Sie die Aufgabe erfolgreich abschließen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, das Internet zu durchsuchen, um aktuelle Informationen abzurufen.
