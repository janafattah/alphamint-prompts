Erstellen Sie einen detaillierten Leitfaden zur Erstellung einer kostenlosen Website. Fügen Sie die folgenden Unteraufgaben in Ihre Antwort ein:

- Überblick über kostenlose Website-Erstellungsplattformen.
- Schritt-für-Schritt-Anweisungen zur Einrichtung einer Website unter Verwendung einer beliebten kostenlosen Plattform (z. B. WordPress.com, Wix oder Weebly).
- Tipps zur Auswahl einer geeigneten Vorlage oder eines Designs für verschiedene Arten von Websites (z. B. persönlicher Blog, Geschäftsseite, Portfolio).
- Grundlegende Anpassungsoptionen (z. B. Ändern des Themas, Hinzufügen von Widgets, Anpassen des Layouts).
- Überblick darüber, wie Inhalte hinzugefügt und organisiert werden (z. B. Erstellen von Seiten, Beiträgen, Hinzufügen von Bildern und Videos).
- Informationen zur Veröffentlichung der Website und zur öffentlichen Zugänglichmachung.
- Grundlegende SEO-Tipps, um der Website zu helfen, von Suchmaschinen gefunden zu werden.

Stellen Sie mir klärende Fragen, bis Sie zu 95 % sicher sind, dass Sie die Aufgabe erfolgreich erledigen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, im Internet zu suchen, um aktuelle Informationen abzurufen.
