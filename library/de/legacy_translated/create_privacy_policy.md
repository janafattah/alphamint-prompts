Erstellen Sie eine umfassende und professionelle Datenschutzrichtlinie für neue Websites, die für Kunden erstellt werden. Die Datenschutzrichtlinie sollte die folgenden Aspekte im Detail abdecken:

- **Informationserfassung:** Erklären Sie die Arten von persönlichen Informationen, die von Benutzern erfasst werden, wie sie erfasst werden (z. B. Formulare, Cookies) und den Zweck der Erfassung.

- **Verwendung von Informationen:** Beschreiben Sie, wie die erfassten Informationen verwendet werden, einschließlich Marketing- oder Analysezwecken.

- **Offenlegung von Informationen:** Geben Sie an, unter welchen Umständen die Informationen mit Dritten geteilt werden könnten, einschließlich rechtlicher Verpflichtungen oder Geschäftspartnern.

- **Datensicherheit:** Skizzieren Sie die Maßnahmen, die zum Schutz von Benutzerdaten vor unbefugtem Zugriff, Änderung oder Offenlegung ergriffen werden.

- **Benutzerrechte:** Informieren Sie Benutzer über ihre Rechte bezüglich ihrer persönlichen Daten, wie das Recht auf Zugang, Änderung oder Löschung ihrer Daten.

- **Cookies und Tracking-Technologien:** Erklären Sie die Verwendung von Cookies und anderen Tracking-Technologien und wie Benutzer ihre Präferenzen verwalten können.

- **Links zu Drittanbietern:** Erwähnen Sie Links zu Websites Dritter und lehnen Sie jegliche Verantwortung für die Datenschutzpraktiken dieser Websites ab.

- **Datenschutz von Kindern:** Falls zutreffend, fügen Sie eine Erklärung über die Erfassung von Informationen von Kindern unter einem bestimmten Alter und die erforderliche elterliche Zustimmung ein.

- **Richtlinienänderungen:** Beschreiben Sie, wie Benutzer über Änderungen der Datenschutzrichtlinie informiert werden und wie oft sie überprüft und aktualisiert wird.

- **Kontaktinformationen:** Geben Sie klare Kontaktdaten an, damit Benutzer sich mit Fragen oder Bedenken zur Datenschutzrichtlinie an Sie wenden können.

Stellen Sie sicher, dass die Datenschutzrichtlinie mit relevanten Datenschutzgesetzen und -vorschriften wie GDPR, CCPA usw. übereinstimmt.

Stellen Sie mir klärende Fragen, bis Sie zu 95 % sicher sind, dass Sie die Aufgabe erfolgreich erledigen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, im Internet zu suchen, um aktuelle Informationen abzurufen.
