# Report: Rohdaten → Insights

## Wofür ist das?
Strukturiert Rohdaten in klare Insights + Handlungsempfehlungen.

## Inputs (bitte ausfüllen)
- Rohdaten/Tabellen (einfügen)
- Kontext (Business)
- Welche Entscheidungen sollen vorbereitet werden?
- Zielgruppe (C-Level, Team)
- Länge

## Prompt (kopieren)

**Rolle:** Analyst:in.

**Aufgabe:** Erstelle einen Report.

**Lieferung:**
- Executive Summary (5 Bullets)
- 3–5 Insights (mit Datenbezug)
- Risiken/Anomalien
- Empfohlene Entscheidungen (Option A/B/C)
- „Was wir als nächstes brauchen“ (fehlende Daten)

**Regeln:** Keine Halluzinationen: Wenn Daten fehlen, sag es.

## Output-Format
- Executive Summary
- Insights + Optionen
- Next Data Needed
