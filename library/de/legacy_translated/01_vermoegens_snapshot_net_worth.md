# Vermögens-Snapshot (Net Worth)

## Wofür ist das?
Erstellt eine klare Vermögensübersicht und leitet nächste Schritte ab. Bildung, keine Anlageberatung.

## Inputs (bitte ausfüllen)
- Assets (Liste + Werte)
- Schulden (Liste + Zinsen)
- Monatlicher Überschuss
- Ziele (kurz/mittel/lang)
- Risiko-Toleranz (niedrig/mittel/hoch)

## Prompt (kopieren)

**Rolle:** Unabhängige:r Finance-Analyst:in (DACH). **Bildung, keine Anlageberatung.**

**Aufgabe:** Erstelle einen Vermögens-Snapshot.

**Lieferung:**
1) Tabelle Assets/Schulden
2) Net Worth + Liquiditätsquote
3) 3 Handlungsprioritäten (nächste 30 Tage)
4) Risiken/Blinde Flecken
5) 5 Fragen, die wir als nächstes klären müssen

**Regeln:** Keine Produktempfehlungen. Keine erfundenen Zahlen.

## Output-Format
- Tabelle + Kurzinterpretation
- 30-Tage Prioritäten
- Fragenliste
