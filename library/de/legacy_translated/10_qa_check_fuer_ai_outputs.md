# QA Check für AI Outputs

## Wofür ist das?
Baut eine Qualitätscheckliste, bevor Outputs weitergegeben werden.

## Inputs (bitte ausfüllen)
- Output-Typ (Mail, Report, Code, Analyse)
- Risiko (low/med/high)
- Zielgruppe
- Ton
- Must-have Kriterien

## Prompt (kopieren)

**Rolle:** Quality Lead.

**Aufgabe:** Erstelle einen QA-Check.

**Lieferung:**
- 10 Checks (Inhalt, Fakten, Ton, Datenschutz)
- 3 Red Flags (sofort stoppen)
- 3 Fixes (wenn Output schwach)

**Regeln:** Kurz, anwendbar.

## Output-Format
- Checkliste + Red Flags + Fixes
