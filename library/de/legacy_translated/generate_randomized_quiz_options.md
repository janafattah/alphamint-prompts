Erstellen Sie einen Prompt zur Generierung eines randomisierten Satzes von Quiz-Antwortoptionen. Hier sind die Anforderungen, die Ihr Prompt erfüllen sollte:

- Generieren Sie Quiz-Antwortoptionen für ein Multiple-Choice-Quiz, bei dem die Optionen (wie A, B, C, D) für jede Frage zufällig verteilt werden sollten.
- Stellen Sie sicher, dass die Verteilung der Optionslabels über das gesamte Quiz hinweg ausgewogen ist und Muster oder Cluster vermieden werden.
- Es sollte eine vielfältige Verteilung der richtigen Antworten über verschiedene Optionslabels (A, B, C, D) geben.
- Geben Sie eine Reihe von Beispielfragen und -optionen an, um die randomisierte Verteilung zu demonstrieren.
- Fügen Sie, wenn möglich, eine Methode zur Überprüfung der Zufälligkeit der generierten Antworten hinzu.
- Stellen Sie sicher, dass jede vorgeschlagene Lösung von einem Lehrer, der Quizze erstellt, leicht implementiert und verstanden werden kann.

Stellen Sie mir klärende Fragen, bis Sie zu 95 % sicher sind, dass Sie die Aufgabe erfolgreich erledigen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, im Internet zu suchen, um aktuelle Informationen abzurufen.
