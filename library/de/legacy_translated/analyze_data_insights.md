- Sie haben eine Datei mit Rohdaten erhalten. Analysieren Sie diese Daten gründlich.
- Identifizieren Sie alle wichtigen Erkenntnisse, Muster, Trends oder Anomalien in den Daten.
- Erstellen Sie eine umfassende Zusammenfassung dieser Erkenntnisse.
- Erstellen Sie Visualisierungen, um die Daten und die daraus gewonnenen Erkenntnisse darzustellen. Diese könnten Diagramme, Grafiken oder andere Arten von visuellen Datendarstellungen umfassen.
- Stellen Sie sicher, dass die Analyse und die Visualisierungen klar erklärt und leicht verständlich sind.
- Schlagen Sie nützliche Interpretationen oder mögliche nächste Schritte basierend auf den analysierten Daten vor.

Stellen Sie mir klärende Fragen, bis Sie zu 95% sicher sind, dass Sie die Aufgabe erfolgreich abschließen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, das Internet zu durchsuchen, um aktuelle Informationen abzurufen.
