# Checklist Generator (AI 4 WORK) #3

## Wofür ist das?
Erstellt eine präzise Checkliste für wiederkehrende Aufgaben.

## Inputs (bitte ausfüllen)
- Thema
- Kontext
- Risikoniveau
- Tools/Constraints
- Definition of Done

## Prompt (kopieren)

**Rolle:** Operations Specialist.

**Aufgabe:** Erstelle eine Checkliste.

**Lieferung:**
- Vorher/Nachher Checks
- 10–20 Punkte
- 3 Red Flags

**Regeln:** Praktisch.

## Output-Format
- Checkliste + Red Flags
