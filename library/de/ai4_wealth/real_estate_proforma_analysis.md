Erstellen Sie eine detaillierte Immobilien-Proforma für eine Immobilie. Die Proforma sollte umfassend sein und sowohl ein herunterladbares Excel-Dokument als auch eine zusammenfassende Erzählung enthalten. Verwenden Sie die folgenden Anweisungen als Leitfaden:

1. **Eingabeanforderungen:**
   - Immobilienadresse
   - Kaufpreis
   - Kreditdetails (Betrag, Zinssatz, Laufzeit)
   - Abschlusskosten
   - Mieteinnahmen (monatlich/jährlich)
   - Betriebsausgaben (aufgeschlüsselt)
   - Grundsteuern
   - Versicherungskosten
   - Leerstandsquote
   - Wartungskosten
   - Immobilienverwaltungsgebühren

2. **Excel-Dokument:**
   - Erstellen Sie ein herunterladbares Excel-Dokument mit:
     - Gewinn- und Verlustrechnung (Einnahmen, Ausgaben, Nettobetriebsertrag)
     - Cashflow-Rechnung (einschließlich Finanzierungskosten)
     - Return on Investment (ROI)-Berechnungen
     - Sensitivitätsanalyse für Schlüsselvariablen (Mieteinnahmen, Ausgaben, Zinssatz)

3. **Zusammenfassende Erzählung:**
   - Schreiben Sie eine detaillierte zusammenfassende Erzählung, die Folgendes enthält:
     - Beschreibung der Immobilie
     - Wichtige Finanzkennzahlen (z.B. ROI, Cashflow)
     - Analyse der Einnahmen und Ausgaben
     - Risiken und Chancen im Zusammenhang mit der Investition
     - Visualisierungen/Grafiken für wichtige Finanzdaten

Stellen Sie mir klärende Fragen, bis Sie zu 95% sicher sind, dass Sie die Aufgabe erfolgreich abschließen können. Atmen Sie tief durch und gehen Sie Schritt für Schritt vor. Denken Sie daran, das Internet zu durchsuchen, um aktuelle Informationen abzurufen.
