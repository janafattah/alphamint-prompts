# Fact-Checking Workflow

## Wofür ist das?
Baut einen einfachen Prüflow für AI-Antworten (ohne Paranoia).

## Inputs (bitte ausfüllen)
- Output-Typ
- Risiko-Level
- Verfügbare Quellen (intern/extern)
- Zeitbudget (2/5/10 min)

## Prompt (kopieren)

Baue einen Fact-Checking Workflow.

**Lieferung:**
- 5-Schritte Check (immer gleich)
- Quellen-Hierarchie (was ist vertrauenswürdig?)
- Red Flags (Halluzination, falsche Sicherheit, fehlende Quellen)
- Prompt: „Check my answer“ (damit das Modell sich selbst prüft)

**Regeln:** pragmatisch, nicht akademisch.

## Output-Format
- Checkliste + Red Flags
- Self-check Prompt
