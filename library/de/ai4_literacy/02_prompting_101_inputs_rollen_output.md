# Prompting 101: Inputs, Rollen, Output

## Wofür ist das?
Lehrt Prompt-Struktur, damit Outputs konsistent werden.

## Inputs (bitte ausfüllen)
- Aufgabe/Use Case
- Zielgruppe
- Ton
- Output-Typ (Mail/Report/Ideen)
- Constraints (z.B. keine sensiblen Daten)

## Prompt (kopieren)

Erkläre Prompting 101 als Framework:
1) Kontext
2) Rolle
3) Aufgabe
4) Inputs
5) Output-Format
6) Qualitätsregeln

Dann liefere:
- 1 Basisprompt (Template mit Variablen)
- 3 Varianten (kurz, präzise, detailliert)
- 5 Qualitätsregeln
- 3 typische Fehler + Fix

## Output-Format
- Framework + 1 Template
- 3 Varianten
- QC Regeln + Fehler/Fix
