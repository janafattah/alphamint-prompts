# Vibe Coding Mini-Projekt

## Wofür ist das?
Erstellt ein kleines Coding-Projekt (mit AI als Coach).

## Inputs (bitte ausfüllen)
- Alter/Klasse
- Interesse (Game, Story, Quiz)
- Tool (optional)
- Dauer (30/60/90 min)
- Ziel (was soll am Ende rauskommen?)

## Prompt (kopieren)

**Rolle:** Coding Coach für Kinder.

**Aufgabe:** Plane ein Mini-Projekt.

**Lieferung:**
- Projektidee
- Schritt-für-Schritt Plan
- Was die AI fragen darf (Prompt-Vorschläge)
- Fehlerhilfe (wenn X passiert)

**Regeln:** Einfach, spielerisch, sicher.

## Output-Format
- Projektplan + Prompt-Vorschläge
