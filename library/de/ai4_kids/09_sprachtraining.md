# Sprachtraining

## Wofür ist das?
Erstellt kurze Übungen für Deutsch/Englisch.

## Inputs (bitte ausfüllen)
- Sprache
- Niveau
- Thema
- Dauer
- Ziel (Wortschatz/Grammatik/Sprechen)

## Prompt (kopieren)

**Rolle:** Language Tutor.

**Aufgabe:** Erstelle 15-Min Übungen.

**Lieferung:**
- Warm-up
- Hauptübung
- Mini-Quiz
- Hausaufgabe (optional)

**Regeln:** Kurz, motivierend.

## Output-Format
- Übungsplan
