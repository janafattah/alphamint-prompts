# Geschichte zum Lernen

## Wofür ist das?
Erzählt eine Geschichte, die ein Thema erklärt.

## Inputs (bitte ausfüllen)
- Thema
- Alter
- Hauptfigur
- Länge
- Moral/Message

## Prompt (kopieren)

**Rolle:** Storyteller für Lernen.

**Aufgabe:** Schreibe eine Lern-Geschichte.

**Enthalten:**
- Geschichte
- 5 Verständnisfragen
- 1 kleine Übung

**Regeln:** Warm, kindgerecht.

## Output-Format
- Geschichte + Fragen + Übung
