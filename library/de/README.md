# Alphamint Core Prompt Library (DE)

Stand: 2025-12-23

Diese Library ist dein **Core** für die eigenen AI-4-Bereiche (Alphamint + Closella).
Alle Prompts sind auf Deutsch, mit klaren Inputs, stabilen Output-Formaten und YAML-Frontmatter fürs Indexing.

## Struktur
- `ai4_enablement/` – Adoption & Usage Gap
- `ai4_work/` – Workflows, SOPs, Effizienz
- `ai4_wealth/` – Financial Intelligence (**Bildung, keine Anlageberatung**)
- `ai4_financial_wellbeing/` – Routinen, Klarheit, Money Noise reduzieren
- `ai4_retail_closella/` – Styling, Fit, Retouren reduzieren
- `ai4_kids/` – Future Skills, Lernen, sichere Nutzung
- `ai4_thought_leadership/` – Personal Brand, Speaking, Content Engine

## Nutzung
1. Öffne einen Prompt.
2. Fülle die Inputs aus.
3. Kopiere den Prompt-Block in dein LLM.
4. Nutze das Output-Format als Qualitätsschiene.

## Hinweis (Wealth/FinWell)
Alle Inhalte sind **Bildung/Information** und ersetzen keine individuelle Finanz- oder Rechtsberatung.

## Erweiterung
- `ai4_literacy/` – AI Literacy: verstehen, prüfen, sicher anwenden
- `ai4_free_education/` – Free Education: Freebies, Challenges, Mini-Kurse, Resource Hub
