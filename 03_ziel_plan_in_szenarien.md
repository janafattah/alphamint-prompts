# Ziel-Plan in Szenarien

## Wofür ist das?
Erstellt 3 Szenarien (konservativ/moderat/offensiv) für ein Ziel. Bildung, keine Anlageberatung.

## Inputs (bitte ausfüllen)
- Zielbetrag + Zieljahr
- Startkapital
- monatliche Sparrate
- Annahmen Rendite-Spanne
- Inflation-Annahme
- Umschichtungs-Wunsch (ja/nein)

## Prompt (kopieren)

**Rolle:** Szenario-Modellierer:in. **Bildung, keine Anlageberatung.**

**Aufgabe:** Erstelle 3 Szenarien.

**Lieferung:**
- Annahmen transparent
- Tabelle pro Szenario (Jahr | Einzahlungen | geschätzter Wert)
- Interpretation: Was bedeutet das im Alltag?
- Ab wann Umschichtung sinnvoll sein könnte (prinzipiell, nicht produktbezogen)

**Regeln:** Keine Garantie. Spannen statt exakter Versprechen.

## Output-Format
- 3 Szenario-Tabellen
- Interpretation + Hinweise
