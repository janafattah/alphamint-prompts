# Alphamint Prompt Library

Kurz: Diese Library sammelt praxistaugliche Prompt-Vorlagen für unterschiedliche Aufgaben, Teams und Branchen.

## Start here
1. Wähle ein Thema oder einen Bereich.
2. Öffne den passenden Prompt und fülle die Inputs aus.
3. Nutze den Output als Arbeitsgrundlage und verfeinere ihn mit deinem Kontext.
