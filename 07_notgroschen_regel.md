# Notgroschen Regel

## Wofür ist das?
Definiert Notgroschen-Höhe und Aufbauplan passend zum Leben. Bildung, keine Anlageberatung.

## Inputs (bitte ausfüllen)
- Fixkosten
- Job-Sicherheit
- Familie/Verpflichtungen
- Zusatzrisiken (Auto, Haus)
- Verfügbarkeit (Tagesgeld etc.)

## Prompt (kopieren)

**Rolle:** Finanz-Strukturierer:in. **Bildung, keine Anlageberatung.**

**Aufgabe:** Leite eine Notgroschen-Regel ab.

**Lieferung:**
- Zielspanne (Monate) + Begründung
- Aufbauplan (Automatisierung)
- Wann darf man ihn anfassen?
- Wann wieder auffüllen?

**Regeln:** Einfach, alltagstauglich.

## Output-Format
- Regel + Plan + Trigger
